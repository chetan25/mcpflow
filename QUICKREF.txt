🎯 WHAT IS MCPFlow?
   Framework for orchestrating Model Context Protocol (MCP) servers.
   Teams can build intelligent agents that discover and invoke tools
   across multiple distributed MCP services.

📦 REPOSITORY
   GitHub: https://github.com/chetan25/mcpflow
   Status: ✅ Production Ready v0.1.0

🚀 QUICK START

   Create MCP Server:
   ──────────────────
   from mcpflow import MCPServerDecorator, tool
   
   @MCPServerDecorator(name="my-tools")
   class MyTools:
       @tool(description="My first tool")
       def my_tool(self, param: str) -> dict:
           return {"result": param}

   Use Configuration:
   ─────────────────
   from mcpflow import Config, MCPRegistry
   
   config = Config.from_yaml("config.yaml")
   registry = MCPRegistry()
   await registry.register_mcp(config.teams[0].mcps[0])

   Mock for Testing:
   ────────────────
   from mcpflow import MCPFixture
   
   fixture = MCPFixture()
   fixture.expect_tool("get_data").returns({"data": []})
   result = fixture.call_tool("get_data", {})
   assert fixture.called("get_data")
